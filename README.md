## 这是什么？

一款以 JRPG 精神为内核的**本地优先个人成长追踪器**。

它把 RPG 角色成长机制移植到真实生活：记录你做的事 → 转化为属性点数 → 积累升级 → 解锁技能与成就 → 与内心的「逆影」对战 → 和 22 位塔罗同伴共度时光 → 用 AI 回望你的成长弧线。

数据完全留在本机；可选开启好友 / 联机同伴等社交功能后，才会上云。

## 核心系统

### 人格五维属性

| 属性 | 默认名 | 代表的行为 |
|------|--------|-----------|
| Knowledge | 知识 | 阅读、学习、思考 |
| Guts | 胆量 | 挑战、运动、意志力 |
| Dexterity | 灵巧 | 技艺、创作、动手能力 |
| Kindness | 温柔 | 关怀、帮助、共情 |
| Charm | 魅力 | 社交、表达、气场 |

五项属性**全部可自定义命名**，初始化引导提供预设风格，也可完全自定义。
属性每升一级可解锁对应技能，技能激活后为该属性的加点提供**额外百分比加成**。

---

### 记录事项（历史记录）

- 用自然语言描述你的行动，**关键词规则**自动识别并建议加点分配
- 手动精细调整每项属性分值（0–5），一次记录可同时影响多项属性
- 标记**重要事件**（⭐），在日历热力图中以琥珀色圆点高亮
- **日历热力图**直观呈现活跃程度，色点大小随当日总点数变化
- 历史记录按年 → 月 → 日三级折叠，支持全文搜索、多维筛选
- 每条记录**长按可删除**，带二次确认保护

---

### 任务系统

- 每个任务绑定属性与奖励点数，完成即自动加点并写入历史
- 四种模式：单次完成 / 每日重复 / 多次计数 / 长期累计
- 支持**按星期过滤**（例如"周一周三才出现"）
- **重要任务**（⭐）置顶 + 琥珀色高亮
- 长按进入编辑模式；已归档可恢复为活跃

---

### 本周目标

- 每周可设定一组目标（2–4 项），目标类型：活动次数 / 任务完成 / 属性点数 / 全属性点数
- 每项目标独立显示进度条，全部达成后出现"领取奖励"
- **完成庆祝弹窗**：彩色粒子爆发 + 选择奖励属性（Lv.3+ 属性 +7，其余 +5）
- **领取确认动画**：扩散光环 + 浮现属性名与加点数
- 过期未完成的目标在下次打开应用时**自动清除**

---

### 成就 & 技能

**成就**达成条件后需要**手动点击解锁**——给自己一个值得记住的仪式感瞬间。

条件类型：连续打卡天数 / 累计总点数 / 属性等级 / 关键词匹配 / 任务完成次数 / 每周目标完成次数 / 击败逆影次数 / 全属性满级 / 同伴亲密度达成

内置 17 项成就，可完全**自由创建、编辑自定义成就和技能**，内置的标题、图标、描述也可修改。

**技能**：每项属性两个内置技能（Lv.3 +20%，Lv.5 再叠加 +10%），多个技能**乘算叠加**。

**馆长的赐福**：初始化时选择的专属技能，为指定属性每次加点额外 +1，可在技能页随时开启/关闭。

## 逆影战场

> *"这就是另一个你——你心中那个被压抑的自我……"*

以你的真实成长数据为基础，AI 生成一个与你属性**完全相反**的"逆影"，通过回合制战斗让你与自身的阴暗面正面交锋。

### Persona（人格面具）

- 通过 **4 道选择题 + 1 道开放题**，AI 从人类历史/神话/文学中为你匹配 5 位 Persona（对应五属性）
- **流式生成**：觉醒动画期间可实时看到 AI 输出的文字从底部浮现
- 每项属性对应一组 Lv.1–5 技能，类型涵盖：**伤害 / 暴击 / 增益 / 减益 / 蓄力 / 治疗 / 攻击增益**
- 可为各属性装备"面具（Mask）"，激活后强化对应属性的攻击效果
- 单属性 Persona 可手动换牌，或用 AI 重新洗牌
- 生成失败可即时重试（换一种说法）

### Shadow（逆影）

- AI 根据你五维属性**自动生成逆影**，各项均为你的"反转投影"
- 每个逆影有独立**弱点属性**（受弱点命中时伤害 ×1.5）
- 共 5 个等级，部分等级带 **Phase 2**（第二形态）
- 首次识破时播放 WARNING 全屏动画
- 可配置战场**时间窗口**（指定时段 + 星期），限时出现增添仪式感

### 战斗系统 v2.0

**状态效果系统** —— 每种属性的 debuff 与攻击增益技能触发**专属副效果**：

| 属性 | Lv4 Debuff | Lv5 攻击增益 |
|---|---|---|
| 知识 | 🎯 猎手标记（受伤 ×1.2，2 回合） | 🔭 洞悉（Shadow 暴击率 −50%） |
| 胆量 | 😱 恐惧（概率跳过行动） | 🔥 狂怒（3 回合 +15） |
| 灵巧 | ☠️ 中毒（每回合 −3 HP，可叠 3 层） | ⚡ 连击（暴击率 +25%） |
| 温柔 | 🌿 镇静（Shadow 攻击 ×0.7） | 🛡️ 护盾（吸收下次伤害 60%） |
| 魅力 | 💋 魅惑（Shadow 概率自伤） | 🎵 共鸣（下次伤害 ×1.8） |

**Shadow AI 决策树**：Shadow 会根据局势做出反应

- 玩家正在蓄力 → **打断**
- 玩家连续 2 次弱点命中 → **警戒**（下回合弱点伤害减半）
- 自身 HP < 30% → **狂化**（攻击 ×1.5，每回合自损 1 HP）
- 玩家 HP < 25% → **追击**（必暴击）
- 玩家身上有 DoT → **嘲讽**
- 默认 → 常规攻击

**战术动作**（战斗面板常驻 3 个按钮）：

- 🛡️ **防御**（0 SP）：本回合所受伤害 ×0.5，回合末恢复 3 SP
- 🔍 **洞察**（2 SP）：弹窗预告 Shadow 下一步意图 + 施加【洞悉】减它 50% 暴击
- ⚡ **All-Out Attack**：连续 3 次命中弱点解锁，消耗全部 SP 释放全属性 Lv.5 技能伤害总和的 60%，带 cut-in 动画

**其他机制**：

- **SP（特殊能量）** 资源管理，技能消耗、回合回复
- **失衡** —— 弱点 / 暴击命中触发，Shadow 跳过一回合反击，此后 4 回合冷却
- 双方状态条实时显示所有 debuff / buff 图标与剩余回合
- HP 扣减与信息框叙事**同步**，先看到"Shadow 发动了攻击"再掉血
- 11 类情境化台词（打断 / 狂化 / Phase2 开启 / DoT 侵蚀 / 被魅惑 / All-Out 预警…）

### 积累与奖励

- 每次击败逆影写入历史记录（来源标记 `battle`）
- 击败记录存档，随击败数累积玩家最大 HP 获得额外加成

## 同伴与塔罗

> *"每一张牌都是一面镜子，映出你与某人之间的回响。"*

### 每日塔罗

- 每天从 78 张塔罗里随机抽取 3 张候选，你选出最终的 1 张
- 牌面决定今日**加成属性与倍率**（替代了旧版的"每日事件"系统）
- AI 根据牌面、正逆位与你当前状态生成**运势解读 + 行动建议**
- 总体运势分为四档：大吉 / 中吉 / 小吉 / 凶

### 中长期占卜

- 针对具体问题抽取 **3 张牌**（过去 / 现在 / 未来），AI 给出深度解读
- 持续 14 天，期间可进行 **1 次追问**（追问也是抽一张新牌 + 流式 AI 回答）
- 自动归档或手动归档，过期自动清理

### 22 位同伴（Confidants）

- 对应塔罗 **22 张大阿卡纳**，每张牌可绑定一位同伴（离线自建 or 在线好友）
- 亲密度 Lv.0–10，关键节点解锁**永久能力**：
  - Lv.2 → **日常加点**：对应属性每次加点额外 +1（乘区外）
  - Lv.4 → **战斗道具**：每 2 天可用一次，回复 HP 或 SP
  - Lv.7 → **永久战斗技能**：该属性技能伤害永久 +1
  - Lv.10 → 圆满，所有能力增强
- **今日互动**：用自然语言记录今日与 Ta 的互动，AI 给出叙事解读 + 相处建议
- **逆流衰减**：连续 3 天无互动 → 亲密点每日 −1（可开关）
- **星移**：每次升级获得 1 次重新生成牌面解读的机会
- **归档与置顶**：情感安全锁（归档前二次确认）
- 自定义头像（本地保留，不上云）

### 谏言 / 残响对话

- 想不通某段关系时，向"知心"倾诉
- 每 7 天可用一次，开启后**有 1 小时**自由聊天，到期消息自动清空
- 消息里用 `@` 提到某位同伴，AI 会参考 Ta 的最近 15 条互动记录作为上下文
- 可随时归档：AI 生成 ≤100 字摘要，移入归档库永久保留
- 支持**打断 AI 输出**、**流式输出**

## AI 成长总结

> *"让我来检视你的记录，客人……"*

每逢**周末、月初、年末**，历史记录页右上角亮起红点提示。

| 特性 | 说明 |
|------|------|
| **流式逐字输出** | 字符实时流入 |
| **一次追问机会** | 总结完成后可向 AI 追问一个问题 |
| **年度总结** | 12 月 31 日偷偷出现专属年度总结入口 |
| **归档保存** | 每份总结可存档 |
| **特殊条目开关** | 可控制"逆影击破 / 每周目标 / 逆流"是否计入总结 |
| **退出确认** | 流式输出中或未保存时退出会二次确认 |

**内置四种风格预设：**

- 🎭 **伊戈尔** — 庄严睿智，将成长比作"灵魂的觉醒"
- 👒 **伊丽莎白** — 热忱好奇，以"外世界观察者"视角解读
- 🎩 **西奥多** — 温文尔雅，注重情感共鸣与内心世界
- 💼 **玛格丽特** — 冷静克制，以精确洞察与简洁建议为主

支持编辑内置预设，亦可添加完全自定义的 System Prompt。

## 在线联机（可选）

> 所有在线功能**默认关闭**；即便注册账号，本地数据仍完整保留在 IndexedDB，云端只作为加密同步与好友中转。

### 账号与同步

- 基于 PocketBase 的轻量账号系统（邮箱 + 密码）
- **全表推送 / 拉取**，头像指纹避免重复上传几百 KB
- **同步黑名单**：可逐表关闭同步（同伴 / 活动 / 任务 / 总结…），核心 3 表受保护
- 冲突检测 + 差异预览对话框

### 好友与祈愿

- 好友申请 / 接受 / 拒绝（3 天冷却） / 解除（7 天冷却） / 21 天自动过期
- **祈愿**：每天对每位好友可发送 1 次（本地 04:00 到次日 04:00）
- 被祈愿的一方战斗 SP **+3**（无战斗状态时静默丢弃）
- 双向互祈 → 系统通知 `prayer_reciprocal`

### 在线同伴（COOP 契约）

- 把好友关系升级为"在线同伴"：双方各自选择对方在自己眼中的塔罗
- 共享亲密度同步：任一方的互动都会让双方卡片同时成长
- 共享事件：看到对方记录了什么，自己这里也追加

### 羁绊之影（联机 Boss）

- 挂在某对 COOP bond 上的专属 Boss，每逢**新月 / 满月**（约 15 天一次）降临
- 每天 18:00–次日 07:00 可攻击，10 天内未击败则撤退
- 共享 HP，独立 Persona；共鸣印记提供 **×1.5 接力加成**
- 击败后双方同伴卡追加"纪念图章"永久保留

## AI 配置

在「**设置 → AI 总结**」中配置：

1. **API 提供商** — OpenAI / DeepSeek / Kimi / Gemini / MiniMax，或填写自定义 Base URL
2. **API 密钥** — 仅存储于本地 IndexedDB，**不上传任何服务器**
3. **连接测试** — 一键验证配置可用性
4. **模型名称**（可选，留空使用各平台默认模型）
5. **风格预设** — 选择或编辑 System Prompt

AI 相关调用（Persona 生成 / Shadow 生成 / 塔罗解读 / 每周总结 / 谏言对话）**全部走同一个 API Key**，流式响应，90 秒超时。

## 数据与隐私

- 默认所有记录、属性、成就、同伴、塔罗历史均存储于**本机 IndexedDB**，不依赖任何云服务
- 导出备份时自动剥离背景图（base64）、API Key、头像等敏感字段
- 导入备份具有**快照回滚保护**：若写入失败自动完整恢复原有数据
- **在线功能完全可选**：可只用云端做设备间同步，不触发任何社交功能
- 同步黑名单允许你精确控制**哪些表不上云**（例如保留同伴记录在本地）
- 自定义头像**始终只存本地**，不随云同步
